Tinfoil Configuration

To connect to the NX Server using Tinfoil, please configure your settings as follows:

  • Protocol: HTTPS
  • Host: nx.domain.at
  • Path: /
  • Title: NX Server
⚠️ Please note: If you encounter any issues or have trouble connecting, please refer to our wiki documentation for additional information. The wiki also provides answers to frequently asked questions regarding the service.
